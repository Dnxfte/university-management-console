# University Management Console

Консольна система управління університетом на Java.

## Мета

Проєкт імітує просту систему для роботи зі студентами, викладачами, курсами та зарахуваннями на курси.

## Функціональність

- CRUD-операції для студентів, викладачів і курсів.
- Зміна статусу студента.
- Зарахування студентів на курси.
- Виставлення оцінок.
- Позначення оплати курсу.
- Перегляд зарахувань студента.
- Виведення транскрипта студента.
- Підрахунок GPA.
- Пошук студента за частиною ПІБ або email.
- Фільтрація студентів за статусом і роком навчання.
- Сортування студентів за ПІБ через bubble sort.
- Фільтрація курсів за викладачем і кількістю кредитів.
- Список неоплачених зарахувань.
- Середній GPA по курсу та семестру.
- Топ-N студентів за GPA.

## Використані теми Java

- класи та об'єкти;
- інкапсуляція;
- наслідування;
- інтерфейси;
- enum;
- конструктори;
- масиви;
- умови та цикли;
- `Scanner` і консольний ввід/вивід;
- `String` API;
- `IllegalArgumentException` для обробки помилок.

Усі дані зберігаються в пам'яті програми у масивах.

## Запуск

Проєкт не використовує Maven або Gradle. Запуск виконується через `javac` і `java`.

```bash
javac -d out $(find src -name "*.java")
java -cp out university.Main
```
